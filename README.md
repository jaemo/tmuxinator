# Tmuxification

Generator for tmux configurations, by default it opens two windows, one
containing vim in your project root.

## Tested on

* Ruby 1.9
* zsh

## Installation

    $ gem install tmuxification

## Usage

    # create a new tmux project

    cd ~/code/my_project
    tmuxification create

    # start the project

    start_my_project
